# @us3r-network/profile

A React component library that provides some of the most commonly used UI components for [Profile Data Model](?path=/docs/model-definition-profile--docs)

@us3r-network/profile is also a library of unstyled components, provides a simpler way to build accessible components with custom styles, while offering the flexibility to drop down to hooks for even more customizability where needed.

## Install

```bash
npm install @us3r-network/auth-with-rainbowkit @us3r-network/profile
```

Where [@us3r-network/auth-with-rainbowkit](?path=/docs/auth-auth-with-rainbowkit--docs) is the peerDependencies of @us3r-network/profile

## Usage

To use the components, you must wrap your application with `Us3rAuthWithRainbowkitProvider` and `ProfileStateProvider`.

<Source code={App} />

`Us3rAuthWithRainbowkitProvider` establishes a session with Ceramic, which is necessary to retrieve information about the currently logged-in user or perform a write operation on the Ceramic node.

`ProfileStateProvider` can assist you in managing and synchronizing the state of all profile components.
