# LoginButton

当需要获取当前登录的用户信息时，或者需要进行 Ceramic node 的写操作时我们需要先与 Ceramic 建立一个会话。

LoginButton 组件内部调用 [@us3r-network/auth-with-rainbowkit](?path=/docs/auth-auth-with-rainbowkit--docs) 的授权流程。

```jsx
import { LoginButton } from "@us3r-network/profile";
```
