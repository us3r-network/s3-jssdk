# UserWallets

UserWallets 是一个 Context 组件，它有一些子组件，用于展示用户拥有的钱包信息，它可以展示当前登录的用户的钱包信息，也可以展示其他用户的钱包信息。

```jsx
import { UserWallets } from "@us3r-network/profile";
```
